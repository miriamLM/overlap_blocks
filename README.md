# OVERLAP BLOCKS

## How to start

1. If you don't have it already, [install Composer](https://getcomposer.org/download/).
2. Run `composer install`
3. See the following usage instructions

## CLI usage
`cd src`

`php index.php input`

- Examples:
    - `php index.php 0h2351v311`
    - `php index.php 0h2311h231`
    
## Requirements
- php >= 7.4
